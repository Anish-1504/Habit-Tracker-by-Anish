💥 HabitZAP! - Ultimate Comic-Style Habit Tracker
HabitZAP! is an advanced, all-in-one tracking solution designed for Anish Patil. It transforms the traditional habit-tracking experience into a high-energy, "Manga-style" interactive quest.

🛠 Tech Stack
HTML5: The entire application is built as a Single-File HTML Architecture. All layout structures, comic-book styling, and interactive logic are contained within a single .html file for maximum portability and ease of use.

🎨 Design & UI
Comic-Book Aesthetic: Features "Bangers" and "Comic Neue" fonts, thick black borders, halftone dot backgrounds, and "POW! BOOM! ZAP!" animations when tasks are completed.

Anime Theme Integration: Built-in options to toggle between Naruto, Demon Slayer, One Punch Man, and other cute anime aesthetics, complete with character banners and themed color palettes.

Dynamic Visuals: Checkboxes turn green with a ✓ when marked, and each row calculates a live 🔥 streak count.

🚀 Key Features
PDF-Faithful Layout:

10 fixed habit rows based on your original template.

Ability to add "N" number of extra rows, functioning like an Excel sheet.


Sleep Tracking: A dedicated section for 5–9 hour sleep logging, featuring a dot-marking system and a line graph inspired by your reference images.

Data Visualization: Includes four chart types:

Per-habit bar charts.

Sleep hours line chart.

Overall completion donut.

Weekly breakdown charts.

Smart Functionality:

Auto-Save: Data persists in the browser so your progress is never lost.

PDF Export: A dedicated button to export your tracker into a high-quality PDF.

Full Responsiveness: Works seamlessly across mobile, tablet, and desktop devices.
